# Hints

These are curator-created TerpVault starter-package notes, not official hints.

<details>
<summary>General approach</summary>

Read each section as its own small text-game mode. The work is interested in how commands, prompts, and expectations change over time, so a command that made sense in one section may not be the right model for the next.

</details>

<details>
<summary>When stuck</summary>

Try the basics first: `LOOK`, `EXAMINE`, checking inventory, rereading the current prompt, and paying attention to any instructions the current section gives you.

</details>

<summary>Official/external hint source</summary>

IFDB links author-hosted hint files for this work. They are not bundled here; consult the IFDB page if you want the original external hints.

</details>
