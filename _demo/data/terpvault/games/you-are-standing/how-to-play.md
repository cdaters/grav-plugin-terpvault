# How to Play

This is a curator-created TerpVault helper note, not official documentation from the author.

Type commands into the interpreter and press Enter. Useful parser commands include:

- `LOOK`
- `EXAMINE [thing]`
- `INVENTORY`
- `SAVE`
- `RESTORE`

The game intentionally moves through different styles of text interaction. If the prompt or available actions seem to change, treat that as part of the design and read each new screen closely.
