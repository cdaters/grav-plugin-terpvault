# How to Play

This is a curator-created TerpVault helper note, not official documentation from the author.

Type commands into the interpreter and press Enter. Useful parser commands include:

- `LOOK`
- `EXAMINE [thing]`
- `INVENTORY`
- `SAVE`
- `RESTORE`

The premise changes the usual parser-game assumptions. Pay attention to senses and timing, and remember that the player character is not an ordinary adventurer.
